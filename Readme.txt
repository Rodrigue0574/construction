🏗️ UpConstruction – Template Bootstrap pour site BTP
📌 Description rapide : UpConstruction est un template léger, épuré et responsive, conçu avec Bootstrap 5.3.7. Il convient parfaitement aux entreprises de construction, architectes, designers d’intérieur/extérieur, et prestataires du secteur BTP.

🔧 Fonctionnalités clés :

Design moderne, clair et personnalisable

Menu mobile off-canvas

Effets d’animation au scroll

Listing intuitif des services

Optimisation SEO

Compatibilité multi-navigateurs

Vidéos et objets responsives

Pas de dépendance à jQuery

Fichiers Sass/SCSS inclus (version pro)

Formulaires PHP/AJAX (version pro)
